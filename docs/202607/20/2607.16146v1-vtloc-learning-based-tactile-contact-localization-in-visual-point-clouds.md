## 摘要
视觉和触觉是机器人感知和操作中互补的模态。视觉提供全局物体上下文，而触觉在接触点提供精确的局部信息。由于需要触觉数据与视觉几何之间的精确空间对齐，整合这些模态进行接触定位（即预测物体表面上触摸的位置）面临着重大挑战。为解决这一问题，我们提出了VTLoc，一种新颖的视觉-触觉框架，利用3D点云作为视觉输入，从触觉读数中定位接触点。VTLoc引入两个关键组件：几何多模态对齐模块，从融合的视觉-触觉特征重建伪点云，并将其与视觉点云对齐以强制执行跨模态的空间一致性；以及迭代定位更新器，利用融合的视觉-触觉特征迭代优化预测的接触位置。在包含100个真实物体的新基准上进行评估，VTLoc通过减少局部到全局的对应歧义，改进了单点触觉接触定位。

## Abstract
Vision and touch are complementary modalities essential for robotic perception and manipulation. While vision provides global object context, touch offers precise local information at contact points. Integrating these modalities for contact localization, i.e., predicting the location of touch on an object's surface, poses significant challenges due to the need for accurate spatial alignment between tactile data and visual geometry. To address this challenge, we propose VTLoc, a novel visual-tactile framework that localizes contact points from tactile readings using a 3D point cloud as visual input. VTLoc introduces two key components: a geometric multi-modal alignment module, which reconstructs a pseudo-point cloud from fused visual-tactile features and aligns it with the visual point cloud to enforce spatial consistencies across modalities; and an iterative localizing updater, which iteratively refines the predicted contact location using fused visual-tactile features. Evaluated on a new benchmark of 100 real-world objects, VTLoc improves single-touch contact localization by reducing local-to-global correspondence ambiguity.

## 1. 核心问题与整体含义（研究动机和背景）
- **研究动机**：机器人的精细操作依赖于视觉与触觉的互补。视觉提供物体的全局形状和上下文，而触觉提供接触点处的精细局部信息。然而，如何准确地将触觉接触点与视觉几何（如点云）进行空间对齐是一大挑战。现有方法（如MidasTouch）依赖物体特定的先验（预计算触觉码本），采样量大、计算成本高，限制了实际应用。
- **核心问题**：实现跨维度匹配——从2D触觉图像定位到3D物体表面上的接触点坐标，并减少局部-全局对应歧义。
- **整体含义**：VTLoc提出一种无需密集物体特定采样的学习框架，直接将触觉读数映射到3D点云上的接触位置，提升单次触觉接触定位的准确性和鲁棒性，为机器人自主感知和操作奠定基础。

## 2. 方法论：核心思想、关键技术细节
- **核心思想**：受人类感知启发（通过记忆重建物体并迭代匹配触觉反馈），VTLoc将触觉图像和原始点云作为输入，通过两个核心模块实现精准接触定位：
  1. **几何多模态对齐模块（GMA）**  
     - 将触觉特征 \(F_t\) 和点云特征 \(F_p\) 融合为 \(F_f\)，然后通过解耦网络重建一个伪点云 \(P_d\)。  
     - 通过最小化 \(P_d\) 与原始点云 \(P\) 之间的**Chamfer距离**（坐标和法线）强制执行几何一致性，增强网络对3D空间的理解。
  2. **迭代定位更新器（ILU）**  
     - 基于门控循环单元（GRU）设计，从初始估计 \(\hat{c}_0 = 0\) 开始迭代更新接触位置：\(\hat{c}_{k+1} = \hat{c}_k + \Delta\hat{c}\)。  
     - 每次迭代输入触觉特征、点云特征和隐藏状态，输出更新量和新隐藏状态。训练时使用指数加权序列损失（\(\gamma=0.9\)）累加所有中间步的误差，最终预测 \(\hat{c}_N\) 作为输出。
- **接触概率热图生成**：预测位置 \(c_p\) 后，通过与候选接触集 \(S_c\) 中的点计算欧氏距离，选出Top-K最近点，再为点云中每个点分配加权距离得分，最终归一化得到0-1概率热图。
- **损失函数**：\(L = L_{seq} + \lambda L_{recon}\)，\(\lambda=1\)。\(L_{seq}\) 为ILU的序列损失，\(L_{recon}\) 为GMA的Chamfer距离重建损失。

- **数据集**：基于ObjectFolder Real构建新基准，包含**100个真实日常物体**（如碗、盘子、勺子、扳手、天鹅等）。每个物体有30-50个接触位置，点云由EinScan Pro HD 2020扫描仪获取并随机采样1024个点（含坐标和法线）。触觉数据由Franka Emika Panda机械臂搭配GelSight传感器采集。

## 6. 主要结论与发现
- VTLoc在单接触定位上显著优于所有baseline，尤其在“非均匀”物体上，ND降低3.11%（vs MCR），Top-5 Acc提升5.66%。
- 添加法线作为输入在非均匀物体上进一步改善性能（Top-1 Acc提升6.91%），但在均匀物体上反而轻微下降，原因是均匀表面缺乏几何判别信息。
- ILU迭代次数以N=16为最佳，过大会导致过拟合（性能下降）。
- GMA模块有效提升几何一致性，加入后ND从22.58%降至20.50%（坐标+法线输入）。
- 多接触定位中，VTLoc在非均匀子集上ND从8.76%（MidasTouch）降至4.85%，优势明显。
- 在未知物体（物体级划分）上也表现出更强泛化能力（Top-5 Acc 32.79% vs MCR 16.39%）。
